배열 관련 내장함수

  • 배열.indexOf("값") : 해당하는 값이 있는 요소의 인덱스 반환, 없다면 -1
  • 배열.concat(배열, 배열, ...) : 배열 합치기
  • 배열.reverse() : 배열의 값을 역순으로 재배치
  • 배열.sort() : 오름차순 정렬
  • 배열.push("값") : 배열의 맨 뒤에 요소 추가 -> 배열의 크기를 반환
  • 배열.pop("값") : 배열의 맨 뒤 요소 제거 -> 제거된 요소 반환
  • 배열.unshift("값") :배열의 맨 앞에 요소 추가 -> 배열의 크기를 반환
  • 배열.shift("값") :배열의 맨 앞 요소 제거 -> 제거된 요소 반환
  • 배열.slice(시작인덱스, 끝인덱스) : 해당하는 인덱스의 요소 추출
  • 배열.join(구분자) : 구분자를 기준으로 요소를 합침
