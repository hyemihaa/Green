영역

div 태그

컨텐츠 영역

content : width, height 속성을 사용하여 가로/세로 길이를 정할 수 있음

padding : 컨텐츠와 테두리 사이에 존재하는 영역(padding, padding-top, padding-bottom,...)

margin : 테두리 바깥에 존재하는 영역(margin, margin-top, margin-left,...)


테두리 
            선택자 {  
                border - 위치 - style : solid / dotted / dashed / double / groove / none, ...
            }

            선택자 { 
                border - 위치 - width : 값;
            }

            선택자 { 
                border - 위치 - color : 색상;
            }

            선택자 {
                border : 두께 스타일 색상;
            }

            선택자 { 
                border - radius : 값;
            }

            선택자 { 
                box - shadow : 가로 세로 흐림 번짐 색상;
            }
