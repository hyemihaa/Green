메인 페이지로 이동

정리
  1. inline : 인라인 레벨 요소로 변경
  2. block : 블럭 레벨 요소로 변경
  3. inline-block : 인라인 레벨 요소로 변경 (컨텐츠 영역의 가로, 세로길이 변경 가능)
  4. none : 요소 숨김 (렌더링은 되지만 보이지 않고 영역을 차지하지 않음)

블록 레벨 요소

블록 레벨 요소 : 영역을 한 줄 다 차지하는 요소

p태그는 블록 레벨 입니다.

인라인 레벨 요소

인라인 레벨 요소 : 사용하는 만큼만 영역을 차치하는 요소 span 태그는 인라인 레벨 입니다.
