선택자

메인 페이지로 이동

정리

선택자 : 특정 HTML 요소를 선택할 때 사용

  1. 전체 선택자(*) : 전체 요소를 선택할 때 사용 
    2.                 * {
                    속성 : 값;
                    속성 : 값;
                }
  2. 태그 선택자 : HTML 파일 내에서 특정 태그들을 모두 선택할 때 사용 
    3.                 태그명 {
                    속성 : 값;
                    속성 : 값;
                }
  3. 클래스 선택자(.아이디명) : 특정 클래스를 가진 요소를 선택할 때 사용 
    4.                 .클래스명 {
                    속성 : 값;
                    속성 : 값;
                }
  4. 아이디 선택자(#아이디명) : 특정 아이디를 가진 요소를 선택할 때 사용 
    5.                 #아이디명 {
                    속성 : 값;
                    속성 : 값;
                }
  5. 속성 선택자(선택자[속성=값]) : 특정 속성의 값을 비교해서 요소를 선택할 때 사용 
    6.                 선택자[속성=값] {
                    속성 : 값;
                    속성 : 값;
                }
  6. 자식 선택자 : 부모 요소의 바로 아래에 있는(자식요소) 요소를 선택할 때 사용 
    7.                 부모요소 > 자식요소 {
                    속성 : 값;
                    속성 : 값;
                }
  7. 후손 선택자 : 부모 요소의 아래 어딘가에 있는(후손요소) 요소를 선택할 때 사용 
    8.                 부모요소 자식요소 {
                    속성 : 값;
                    속성 : 값;
                }
