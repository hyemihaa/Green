사용자로부터 입력을 받을때 사용

  1. *text : 사용자로부터 문자를 입력받을 때
  2. *password : 사용자로부터 비밀번호를 입력받을 때 사용
  3. search, url, email : text와 동일하지만 세분화된 기능을 제공



  4. textarea : 여러줄을 입력받을 수 있는 태그
  5. number : 숫자를 입력받을 수 있는 태그(스핀박스 생성)
  6. range : 슬라이드바를 사용한 숫자를 입력하는 태그
  7. 날짜를 입력받는 태그들
    • date :
    • month :
    • week :
    • time :
    • datetime-local :
  8. radio : 하나만 선택해야할 때 사용
    9. 선택안함 남자 여자
  9. checkbox : 여러개 선택이 가능할 때 사용
    10. 선태안함 남자 여자
