메인 페이지로 이동

정리
  1. br 태그는 문자를 줄바꿈(개행)

  2. hr 태그는 가로로 선을 그어 상/하단을 구분
  3. h 태그는 제목을 입력할때 주로 사용 (숫자가 작을수록 폰트가 커지고, 숫자가 클수록 폰트가 작아짐)

    4. h1 태그

    h6 태그
  4. strong, b태그는 문자를 굵게 표시하여 강조
    5. strong 태그는 스크린 리더 드의 사용자에게 중요한 내용이면 강조해서 읽는 기능까지 있음
    b 태그는 문자를 굵게 표시
  5. small 태그는 기본 문자보다 작은 글씨를 표현
    6. 원래글씨 vs small 태그
  6. u 태그는 글자에 밑줄을 표시
  7. s 태그는 글자에 취소선을 표시
  8. p 태그는 글의 단락을 나타내는 태그
  9. sup 태그는 위첨자
  10. sub 태그는 아래첨자
